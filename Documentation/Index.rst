----

The autoloader extension creates with the help of configurable and implementable loaders, the global configurations, database schemas, templates etc. necessary for TYPO3.
The goal of this extension is an intelligent "convention over configuration" pattern, in order to reduce the number of the standard necessary configurations in e.g. ext_tables, ext_localconf.
